首頁 產品 報表 新聞 關於 規範與協議 GitHub 嗨，很高興見到你。 卡莉絲伊繁星不是一間公司。

相較於稱呼自己是間「科技公司」，我們的運作模式更像是所「學校」。雖然卡莉絲伊繁星提供網際網路服務與電子科技產業的相關產品，但我們更注重人們是否能在開發的過程中有所收穫並學以致用，而我們也十分地重視一項產品是否能夠達成社會的期許並希望能藉此改變某些人的生活，協助大眾更向自己的目標邁進；這也正是伊繁星最高協議被創造的原因。

我們的願景與目標並非一成不變，但也不是盲目地跟隨潮流。認真感受社會的事物，找出能夠改進的點即是卡莉絲伊繁星的目標。我們的願景是找處人與人之間的聯繫方式並建立出一個友善的社群環境。使命則是竭盡所能地改變這個世界的每個角落；就算目前不能，也永不放棄。

與其執著在如何領先對手，我們更希望能夠相互扶持、合作並且提昇整個市場的發展與社會層次。

 打造適合內容創作者的環境，推動人們之間的社群交流。

正如先前所說，我們的願景是能夠打造一個被大眾接納的社群環境且人與人之間都能夠友善地在其中相互交流。同時應該以最低限度（如：不違反國家法律的情況下）盡可能地保障創作者對於建立自己熱衷事物的權利。這意味著沒有主動審查、沒有人可以干涉他們的思考能力，而唯一能夠限制他們的就僅有想像能力。猶如世界人權宣言那樣地簡潔扼要：

我們都有權利自己做決定，去想我們所喜歡的東西，去說我們所想到的，並且與其他人分享我們的觀念。

人權第十九條：表達的自由

除了保障創作者的自由；我們也盡可能地創造出更多交流與推廣的機會，讓任何人努力的結晶都能夠隨時隨地被注意、欣賞並作為一個靈感的泉脈提供給更多人自由發揮的靈感。我們也試圖在其中尋找是否有方式能夠以此活化整體市場脈絡，讓專長成為真正具有價值的事物並且學習如何互相尊重。

「你只能死一次，一定要死的轟轟烈烈。」

沒有人可以做一件事情直到永遠，而一個人的生命也不是永生。死亡給予了我們試著活下去的動力，在那之前都應該竭盡所能的向著自己的夢邁進。

卡莉絲伊繁星就像是公車的轉運站，我們不希望有人呆滯在這裡不走，因為我們更希望大眾可以透過這裡提昇自己然後找到一個更好的歸屬。盡可能地運用我們所提供的資源然後朝下個地方前進才正是我們希望能看到的。將自己的生命活得淋漓盡致才是最重要的；不要將剛開始的序章放在頁尾。

 科系 支援科 作為在多方科系之間的聯繫橋樑並協助對外溝通。

整合在不同科系之間的資訊傳遞並統整出一個易懂的概要，在進行交流時能夠理解雙方的需求並得到一個結果。處理來自世界各國與不同語系的聯繫、詢問電子郵件並且向對應科系請求協助、整合處理方式後將回應以請求語言回覆。

 Support 應用科 研發硬體並整合相關驅動程式與軟體以開發裝置。

與程式相互輔助並以硬體為主的發展科系。除錯、研發硬體，撰寫相關驅動程式並透過結合軟體以實現如：行動與智慧型裝置、監測設備、可穿戴裝置。透過 3D 列印技術打樣並組裝測試模型。建立一個能夠自我運作的系統或中心裝置，並與其他輔助子裝置進行連動。

 Application 決策科 替整體營運規劃做出妥善決策並構思全新道路。

隨時觀察並了解目前世界發展的現況，試圖找出最根本的原因加以改進得到革新的結果並將其應用於卡莉絲伊繁星的產品發展。規劃妥善的社群決策與輔助、建立創作者友善的環境，促進整體市場的發展。跳脫原先的框架建立出新興產品作為輔助與道路開導。

 Decision 系統科 建置伺服器與雲端系統並維護、管理服務的運行。

架設本機或雲端租賃的網際網路服務與物件、多媒體存儲伺服器與網路、防火牆的基本配置。確保伺服器在執行時的穩定性與備用可行性，同時配置良好的負載平衡器並最佳化系統效能。

設置雲端配額警報並隨時監察系統負荷狀況。串連雲端服務與打造無伺服器（Serverless）或可供微服務佈署的相關環境。

 System 安全科 測試整體環境、邏輯並確保程式資訊安全。

負責測試軟體上的邏輯功能並試圖破解與找出其漏洞以改進缺失。同時確保資料的保存安全與網路邏輯圖的布局，令網際網路的外內層級領域在歸屬時變得更加明確。

設置資料庫與多媒體的本機與雲端備份並配置 RAID 與 NAS 存儲系統的相關設置。測試資料的回溯性和重建時的可用性是否正常。

 Security 音訊科 替應用程式或遊戲建立符合的音效、背景音樂和專輯樂曲。

以實際音聲錄製並替遊戲的各種生動場景建立環境音場效果。透過電腦編曲軟體建立應用程式的相關音效或是以虛擬樂器與 MIDI 鍵盤、琶音器與虛擬鼓手和相關混音器譜曲出電子樂曲、循環片段音樂。

在合作企劃中替演唱者或特別人物建立與其風格相符的專輯樂曲、開場與結束音樂。

 Audio 程式科 依照需求架構核心服務並撰寫商業邏輯與演算法。

透過程式語言實現商業邏輯並以網際網路作為主要視圖將內容呈現給位於世界各地的大眾。建立以 Webview 為主的應用程式提供給任何裝置、雲端串流使用。建構網路服務與相關伺服器開發環境並研究如何以最高效率處理來自全球的資訊傳遞負載量。

除了網路服務外——亦會透過遊戲引擎建立平面、立體的大型角色扮演遊戲。透過類神經網路與機器學習進行姿態捕捉的相關應用，同時協助發展虛擬實況主（如：Virtual YouTuber）的相關產業。研究並建立開放原始碼套件供社群應用。

 program 設計科 透過靈感構思出商標與圖示和 3D 立體模型；建立淺顯易懂的圖文宣傳。

使用向量繪圖軟體設計獨樹一格的圖示與商標並透過相關軟體建立一個令人討喜且帶有良好使用體驗的應用程式互動介面。同時將複雜度最小化並設計出一個簡潔且親近易人的廣告文宣。

協助在遊戲或是企劃專案建立物體與人物的 2D 或 3D 立體模型、CG 並進行生活與世界場景的實地勘查。透過相機採樣需求素材與材質貼圖並應用至虛擬世界。

 Design 資訊科 統整項目與專案資訊並概算收入與支出和未來動向。

了解各科系的大略性質與相關知識並在一個企劃的起始週期規劃出所需資料，替不同科系彙整出其作業研究資料、相關資訊連結、文章供開發參考。追蹤開發進程並且依照不同場景提出對應的解決資料。

以最高效益的方式估算開銷與開發時程的對應發展。計算收入與統整明細並以此作為未來的規劃基礎。
